About MRPT

The Malaria Risk Prioritization Tool helps authorities identify high-risk areas for targeted interventions, optimizing resource allocation for malaria prevention.

Version: 2.0

Legal Information

Data Privacy Notice

This tool collects usage data to improve functionality and user experience. The data collected includes:

  • Uploaded data file structures (not content)
  • Interaction patterns and analysis requests
  • Feature usage statistics
  • Error reports

No personal information is collected beyond session identification. All data is anonymized and used solely for improving the tool's functionality.
