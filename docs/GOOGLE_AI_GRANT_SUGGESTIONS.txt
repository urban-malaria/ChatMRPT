  The project integrates several core technologies:

  Earth Observation Platforms (Google Earth Engine, Sentinel-1, Landsat,
  Digital Elevation Models, flooding layers): Provide the computational
  engine for processing high-resolution satellite imagery to extract
  environmental indicators such as flood risk, surface water presence,
  and vegetation indices.

  Google OpenBuildings: Supplies large-scale building footprint data that
  enables estimation of settlement density, urban structures, and housing
  distribution to identify informal and slum settlements.

  Python/Flask Web Application: Powers the ChatMRPT platform — a full-stack
  web application using Flask for the backend, React for the interactive
  frontend, and libraries including Pandas, GeoPandas, Scikit-learn,
  Matplotlib, and Seaborn for data processing, statistical modeling, and
  geospatial visualization.

  OpenAI GPT-4o (Large Language Model): Provides the conversational AI layer
  that translates complex analytical outputs into accessible explanations
  and decision-support prompts. The LLM interprets user queries in natural
  language, routes them to appropriate analysis tools, and generates
  context-aware responses that guide government users through risk
  assessment and prioritization workflows.

  Cloud Infrastructure (Amazon Web Services): Hosts the ChatMRPT application
  on redundant EC2 instances behind a load balancer with CloudFront CDN,
  supporting scalable data processing, Redis-based session management, and
  secure access for government users across multiple states.

  R/Shiny: Powers the original Malaria Reprioritization Tool (MRPT) web
  application for statistical processing and map generation, which
  ChatMRPT builds upon and extends.
